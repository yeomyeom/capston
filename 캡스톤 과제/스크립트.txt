웹 크롤링 또는 웹 파싱이라고 용어를 통일해서 말합시다.

1장
크롬 확장 프로그램이란?
크롬 브라우저 안에서 기능을 추가해 주는 프로그램입니다.
html, css, javascript 같은 웹 기술과 구글 크롬 확장 API를 이용해 개발이 가능합니다.
chrome://extensions/로 개발자 모드를 키면 개발이 가능합니다.

2장
manifest.json
(안드로이드 앱 개발에 사용되는 xml 파일처럼 크롬 확장프로그램에서 사용하고자 하는 컴포넌트 명시하는 파일입니다.)
payload.js(웹 크롤링을 수행하는 메소드가 들어있습니다.)
popup.js(사용자에게 보여주기 위한 팝업창입니다.)
이렇게 총 3가지 파일로 구성되어 있습니다.

3장

크롬에서 f12를 누르면 현재 페이지의 HTML 파일을 읽어 올 수 있는데 지금 보이는게 다 입니다.
